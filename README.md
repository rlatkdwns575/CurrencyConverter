# USD/KRW 장기 투자 의사결정 모델

원/달러(USD/KRW) 환율과 거시경제·금융시장 데이터를 활용해  
**장기 투자 관점에서 달러 매수 여부를 판단하는 Transformer 기반 모델**을 구축한 프로젝트입니다.

본 프로젝트는 단순히 환율 값을 예측하는 것이 아니라,  
**“지금 매수했을 때 6개월 보유 후 연환산 7% 이상 수익을 기대할 수 있는가”**를 분류하는 방식으로 설계했습니다.

---

## 1. 프로젝트 개요

### 프로젝트 목표
- USD/KRW 환율 관련 데이터를 기반으로 달러 매수 여부 판단
- **최소 6개월 이상 보유**를 전제로 한 장기 투자 의사결정
- 단순 예측이 아니라 **실제 투자 행동으로 연결 가능한 분류 모델** 구축
- 이후 Streamlit을 이용해 결과를 직관적으로 시각화

### 핵심 아이디어
이 프로젝트는 “환율이 오를까?”를 묻는 대신,

> **“오늘 달러를 매수하면, 6개월 뒤 연환산 수익률이 7% 이상이 될 가능성이 충분한가?”**

를 기준으로 학습합니다.

즉,  
- 입력: 과거 일정 기간의 시계열 데이터  
- 출력: 매수(1) / 비매수(0)

형태의 **이진 분류 문제**로 접근했습니다.

---

## 2. 데이터 구성

본 프로젝트는 여러 소스의 데이터를 통합하여 사용합니다.

### 사용 데이터
- **FRED**
  - 미국 금리
  - 미국 CPI
  - 미국 실업률
  - 산업생산
  - 기타 거시 지표
- **ECOS (한국은행)**
  - 한국 거시경제 지표
  - 금리, 경상수지, 통화량 등
- **환율 및 시장 데이터**
  - USD/KRW
  - 기타 파생 feature 생성용 시계열 데이터

### 데이터 전처리
- 날짜 기준 정렬
- 수치형 변환
- 결측치 처리: `forward fill` 우선, 필요 시 `back fill`
- 학습 타깃 생성:
  - 6개월 뒤 환율 기준 수익률 계산
  - 이를 연환산 수익률로 변환
  - `연환산 수익률 >= 7%` 이면 `target = 1`
  - 아니면 `target = 0`

---

## 3. 타깃 정의

### 보유 기간
- 약 **6개월**
- 거래일 기준 `126일`

### 목표 수익률
- **연환산 7%**

### 타깃 생성 방식
```python
future_fx = df["usd_krw"].shift(-126)
hold_return = future_fx / df["usd_krw"] - 1.0
ann_return = (1.0 + hold_return) ** (365.0 / 126) - 1.0
target = (ann_return >= 0.07).astype(int)
